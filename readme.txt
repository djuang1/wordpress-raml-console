Provides a way to embed a RAML Console in a page or post. Plugin adds a button to the Editor as well as a shortcode.
 
== Description ==
 
This plugin provides a way to embed a RAML Console on a WordPress page or post. The RAML Console is generated from the Root RAML URL that is created when you setup an API Portal in the Anypoint Platform.
 
== Installation ==
 
1. Unzip and copy package to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit or create a new page where you want to display the RAML Console
4. Click on the 'M' button
5. Paste the Root RAML URL and click OK
6. Update or Publish page to see RAML Console. 

== Frequently Asked Questions ==
 
= Where do I get the Root RAML URL =
 
You can get the Root RAML URL in the MuleSoft Anypoint Platform when you create an API with RAML and generate a Portal. You can copy the URL from the API Reference page.
